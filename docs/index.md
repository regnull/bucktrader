# Bucktrader

Bucktrader is an event-driven algorithmic trading framework for backtesting and live trading.

Use this documentation in this order:

1. Read the high-level model in `Overview`.
2. Follow `Getting Started` and `Quick Start`.
3. Dive into guides for strategies, indicators, broker, and data feeds.
4. Use `API Reference` for class/method details from docstrings.

## Documentation Map

- **Getting Started**: environment setup and project installation.
- **Quick Start**: first runnable strategy.
- **Core Guides**: architecture and subsystem specifications.
- **Optimization**: parameter sweeps and result handling.
- **Example Strategies**: practical templates to extend.
- **API Reference**: auto-generated module/class docs.
