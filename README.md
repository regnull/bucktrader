# Bucktrader

Event-driven algorithmic trading framework for backtesting, paper trading, and live trading.

## Install

```bash
pip install -e ".[dev,docs]"
```

## Run Tests

```bash
pytest
```

## Documentation

- Build docs locally:

```bash
mkdocs serve
```

- Main docs sources:
  - `docs/getting-started.md`
  - `docs/quickstart.md`
  - `docs/specs/01-overview.md`
  - `docs/examples.md`
  - `docs/api/`
